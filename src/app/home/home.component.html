The XLE UD+Glue web application

This page presents several resources used for exploring LFG's glue semantics in combination with Universal Dependencies as annotated by Stanza. Concretely, it presents the Glue semantics workbench and the LiGER rewrite system . As all of these resources, this frontend is open source. It is available at: XLE UD+Glue client

Quick start

  • Download the UD+Glue repository
  • Install Docker
  • Navigate to the Docker folder in the repository on your computer in a terminal and execute
    docker compose up --build
