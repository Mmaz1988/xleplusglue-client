The XLE+Glue web application

This application presents several resources used for exploring LFG's glue semantics. Concretely, it presents the Glue semantics workbench and the LiGER rewrite system . It also serves as a revised frontend for the original XLE + Glue. As all of these resources, this frontend is open source. It is available at: XLE+Glue client

Using the system

This system is designed for explorative analysis of formal linguistic annotation tools. It provides several interfaces:
  • Demo - a simple interface for testing the LiGER rewrite system and the Glue semantics workbench.
  • Regression testing - An interface for incremental testing of XLE+Glue grammars.
  • Inference - An interface for testing the inference capabilities of XLE+Glue.
