The XLE UD+Glue web application

This page presents several resources used for exploring LFG's glue semantics in combination with Universal Dependencies as annotated by Stanza. Concretely, it presents the Glue semantics workbench and the LiGER rewrite system . As all of these resources, this frontend is open source. It is available at: XLE UD+Glue client

Quick start

  • Download the UD+Glue repository
  • Install Docker
  • Download the English Stanza model and put it in the corresponding folder (see below)
  • Navigate to the Docker folder in the repository on your computer in a terminal and execute
    docker compose up --build

Installing Stanza

If not already there, create a directory named models in the Stanza directory.

Download the English model from Stanza English model.

Download the resources file from Stanza resources.

Put the downloaded files in the models folder. The structure should look like this: 

Stanza/
 └── models/
     ├── en/          ← contents of the English model in the default.zip file
     └── resources.json    ← contents of the resources_1.10.0.json file

The English model should be stored in the Stanza/models/en/ directory unzipped, and the resources file should be in the Stanza/models/ directory.
